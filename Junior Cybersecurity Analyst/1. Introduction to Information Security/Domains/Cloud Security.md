
All the measures taken to keep your data and applications safe when they're stored in the cloud

`shared responsibility model` --> Facility provides the security of the infrastructure and client from the data and applications. If not:

`Misconfigurated Cloud Storage -> Unathorized acces -> Insecure APIs -> Data breach

`Account Hijacking` --> Steal your credentials to gain control to accounts and data

### Key Areas Of Cloud Security

- `Data protection` --> encription
- `Identity and Acces Managment (IAM)` --> 2FA ...
- `Networking Security` --> firewalls, vpn, monitoring ...
- `Compliance[1] and governance` --> follow laws and regulations involving data handling, network...
