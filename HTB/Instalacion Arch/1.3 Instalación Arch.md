
Una vez conectamos el USB, encendemos el dispositivo y presionamos la tecla para entrar en el `Boot Manager` (nos permite escoger que dispositivo arrancar), en mi caso F12 y la unidad Flash USB.

una vez escogida se nos redirige al GRUB para escoger el arranque, en este caso el de instalación de Arch.

`UEFI`
- Unified Extensible Firmware Interface
- Es el reemplazo moderno del BIOS.
- Controla cómo se arranca tu ordenador antes de cargar el sistema operativo.
- Ventajas sobre BIOS:
    - Arranque más rápido
    - Soporta discos grandes (>2 TB)
    - Puede usar **particiones GPT**
    - Mejor seguridad (Secure Boot)

`GRUB`
- GRUB = GRand Unified Bootloader
- Es un **programa que gestiona el arranque de sistemas operativos**.
- Si tienes **Mint + Arch**, GRUB te permitirá elegir entre Mint y Arch al arrancar.
- GRUB se instala dentro de la **partición EFI** si estás en UEFI.
